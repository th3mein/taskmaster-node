# Customer ticket manager

A small Node REST Api | CRUD | Authentication and Authorization with JWT

---

## Stack

- Node
- Express
- MongoDB
- Mongoose
- Typescript

### Spin up a server

- #### Development

      npm run dev

- #### Build and start

      npm run build
      npm start

- #### Environment Variables
      DATABASE_URI=
      ACCESS_TOKEN_SECRET=
      REFRESH_TOKEN_SECRET=
